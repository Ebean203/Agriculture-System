## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Ebean203/Agriculture-System.git
   ```

2. Set up your XAMPP environment:
   - Place the project in your `htdocs` folder
   - Start Apache and MySQL services

3. Import the database:
   - Create a new database named `agriculture_system`
   - Import the `agriculture_system.sql` file

4. Configure the database connection:
   - Open `conn.php`
   - Update the database credentials if needed

5. Access the system:
   - Open your browser
   - Navigate to `http://localhost/agriculture-system`

## Project Structure

- `index.php` - Main dashboard and entry point
- `conn.php` - Database connection configuration
- `farmer_regmodal.php` - Farmer registration modal form
- `record_yield.php` - Yield recording functionality
